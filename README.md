# HPC_Lab0_MatMul

# Реализация
**CPU**

Реализация выполнена с использованием трёх вложенных циклов:

* внешний цикл по строкам

*	внутренний цикл по столбцам

*	цикл суммирования

Особенности:

*	последовательное выполнение

*	высокая вычислительная сложность O(N³)

*	низкая производительность на больших данных

**GPU**

Реализация выполнена с использованием CUDA через библиотеку Numba

# Распараллеливание

Каждый поток GPU вычисляет один элемент результирующей матрицы:

 - 1 поток → 1 элемент
- потоки работают независимо
- используется двумерная сетка потоков
